# shruglang
## Commands:
| NAME | MEANING |
|---|---|
| {         | as long as |
| }         | endif |
| '         | input |
| "         | print |
| •-•        | random default |
| \         | retreat one cell |
| /         | advance one cell |
| +         | add one |
| -         | subtract one |
| #{} | time limit | between curly brackets goes the instructions (only + -) to an amount of time|
| @         | reset all, go to cell 0 |
