# shruglang
## Commands:
| NAME | MEANING |
|---|---|
| #(cell number) | iterations |
| {         | as long as |
| }         | endif |
| '         | input |
| "         | print |
| ,(ip),    | fetch |
| •-•        | random default |
| \         | retreat one cell |
| /         | advance one cell |
| +         | add one |
| -         | subtract one |
| $$(time)$$ | time limit |
| @         | reset all, go to cell 0 |
